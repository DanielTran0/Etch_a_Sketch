# Etch-a-Sketch
Draw Pixel Art   

[DEMO](https://danieltran0.github.io/Etch_a_Sketch/)  
Features include grid size from 1x1 to 100x100, custom colours, random colours, and grey to black (each pass makes the pixel 10% darker).
